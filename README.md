# Real-Time Image Transmission System

Bu proje, WebRTC teknolojisini kullanarak tarayıcılar arasında yüksek performanslı, düşük gecikmeli görüntü ve veri aktarımı sağlamaktadır.

## Özellikler
- **WebRTC Entegrasyonu:** Peer-to-peer (eşler arası) doğrudan veri transferi.
- **Düşük Gecikme:** Gerçek zamanlı analiz ve görüntüleme için optimize edilmiş akış.
- **Görsel Derinlik Analizi:** Gelen görüntülerin gerçek zamanlı olarak derinlik ve mesafe analizi için altyapı.

## Teknolojiler
- JavaScript (WebRTC API)
- HTML5 / CSS3
- Node.js (Signaling server için uygun yapı)

## Kullanım
1. `index.html` dosyasını tarayıcınızda açın.
2. Kamera izni verin ve "Kamerayı Başlat" butonuna tıklayın.
